# Encapsulation (은닉)

>
>  접근 지정자를 통해 구현 (public, protected, private)
>


|    제한자    | 용도  |  <  |  <  | 접근 가능 범위 |   <    |     <     |  <  |
| :-------: | :-: | :-: | :-: | :------: | :----: | :-------: | :-: |
|           | 클래스 | 생성자 | 멤버  |  같은 클래스  | 같은 패키지 | 다른 패키지 자손 | 전체  |
|  public   |  o  |  o  |  o  |    o     |   o    |     o     |  o  |
| protected |     |  o  |  o  |    o     |   o    |     o     |     |
|  package  |  o  |  o  |  o  |    o     |   o    |           |     |
|  private  |     |  o  |  o  |    o     |        |           |     |
