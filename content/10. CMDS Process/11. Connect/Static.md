## Static

>[!특징]
>객체를 생성하지 않아도 호출이 가능함

##### 생성 가능한 위치
- 변수
- 메소드
- 내부클래스
- 블럭

|          |            인스턴스 변수             |             static 변수              |
| :------: | :----------------------------: | :--------------------------------: |
| 호출 가능 시점 | Heap에 존재하기 때문에 객체를 생성해야지 호출 가능 | Static 영역에 따로 저장되므로 객체 생성 전에 호출 가능 |
|  생성 개수   |           객체당 하나 생성            |            class당 하나 생성            |
=> static이 들어간 메소드는 static 변수만 참조 가능
