## 4지선다 문제
1. 람다 표현식이 아닌 메서드 참조는 다음 중 어떤 경우에 사용될 수 있는가?
	1. A. 인스턴스 메서드 호출
	2. B. 정적 메서드 호출
	3. C. 생성자 호출
	4. D. 모두 해당
2. 특정 객체의 인스턴스 메서드 참조에서 첫 번째 파라미터는 무엇인가?
	1. A. 메서드 이름
	2. B. 메서드 호출 객체
	3. C. 파라미터 타입
	4. D. 리턴 타입
3. 다음 코드에서 'names.forEach(System.out::println);'은 어떤 종류의 메서드 참조인가?
	1. A. 인스턴스 메서드 참조
	2. B. 정적 메서드 참조
	3. C. 생성자 참조
	4. D. 함수형 인터페이스 호출

## 정답
1. D
2. B
3. A
