## Flood Fill
- 다차원 배열의 어떤 칸과 연결된 영역을 찾는 알고리즘
- 시작점으로부터 연결된 영역을 찾는 알고리즘
	- 4방탐색
		- `dx = {1, 0, -1, 0}`
		- `dy = {0, 1, 0, -1}`
	- 8방탐색
		- `dx = {1, 1, 0, -1, -1, -1, 0, 1}`
		- `dy = {0, 1, 1, 1, 0, -1, -1, -1}`
